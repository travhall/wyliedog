# UI Component Library

Welcome to our UI component library! This is a collection of reusable components built with:

- React + TypeScript
- Tailwind CSS
- Radix UI primitives
- Lucide React icons

## Getting Started

```tsx
import { Button } from "@/components/ui/button";
import { Card } from "@/components/ui/card";

export default function App() {
  return (
    <Card>
      <Button>Click me</Button>
    </Card>
  );
}
```

## Component Categories

Our components are organized into the following categories:

- **Core** - Basic building blocks (Container, AspectRatio, etc.)
- **Data Display** - Components for displaying data (Table, Avatar, Badge, etc.)
- **Feedback** - User feedback components (Toast, Alert, Progress)
- **Forms** - Form controls and inputs (Input, Button, Select)
- **Navigation** - Navigation components (Menu, Breadcrumbs, Pagination)
- **Overlay** - Overlay components (Dialog, Sheet, Tooltip)
